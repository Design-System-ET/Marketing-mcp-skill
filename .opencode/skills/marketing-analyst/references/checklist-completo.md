# Checklist de Proyecto — Marketing Analyst

## FASE 1 — DIAGNÓSTICO ESTRATÉGICO
- [ ] 1.1 Entrevista de descubrimiento realizada y brief documentado
- [ ] 1.2 Análisis de producto/servicio completado
- [ ] 1.3 Perfil de Cliente Ideal (ICP) definido
- [ ] 1.4 Identidad de marca documentada
- [ ] 1.5 Objetivos SMART definidos
- [ ] 1.6 Matriz FODA completada

## FASE 2 — INVESTIGACIÓN DE MERCADO
- [ ] 2.1 Matriz competitiva con 3-5 competidores
- [ ] 2.2 Calendario de estacionalidad y eventos Uruguay
- [ ] 2.3 Benchmark de precios vs competidores
- [ ] 2.4 Lista de palabras clave priorizada

## FASE 3 — AUDITORÍA DE ACTIVOS DIGITALES
- [ ] 3.1 Informe de salud del sitio web
- [ ] 3.2 Diagnóstico de redes sociales
- [ ] 3.3 Tracking y píxeles verificados
- [ ] 3.4 Reporte SEO con quick wins
- [ ] 3.5 Diagnóstico de email marketing (si aplica)

## FASE 4 — PRESENCIA DIGITAL
- [ ] 4.1 Sitio web optimizado o creado
- [ ] 4.2 Perfiles sociales optimizados
- [ ] 4.3 Stack analítico configurado

## FASE 5 — ESTRATEGIA Y PLAN DE MEDIOS
- [ ] 5.1 Propuesta comercial y UVP definida
- [ ] 5.2 Plan de medios con presupuesto por canal
- [ ] 5.3 KPIs y proyecciones documentadas

## FASE 6 — SETUP TÉCNICO Y CREATIVIDADES
- [ ] 6.1 Píxeles y tags funcionando
- [ ] 6.2 Conversiones de Google Ads configuradas
- [ ] 6.3 Feed de productos activo (si aplica)
- [ ] 6.4 Audiencias creadas y listas
- [ ] 6.5 Creatividades diseñadas y aprobadas

## FASE 7 — PILOTAJE Y TESTEO
- [ ] 7.1 Campaña piloto lanzada (7-14 días)
- [ ] 7.2 Pruebas A/B en curso
- [ ] 7.3 Análisis de resultados completado

## FASE 8 — LANZAMIENTO Y ESCALADO
- [ ] 8.1 Campañas escaladas gradualmente
- [ ] 8.2 Marketing de contenidos activo
- [ ] 8.3 Reportes automáticos configurados
- [ ] 8.4 Ciclo de optimización semanal establecido
