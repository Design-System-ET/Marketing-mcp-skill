# Pendientes de configuración - Global Infinity Marketing

## MCPs que requieren credenciales

- [ ] **Mercado Libre** — Crear app en developers.mercadolibre.com para obtener CLIENT_ID y CLIENT_SECRET
- [ ] **Google Ads** — Configurar proyecto en Google Cloud + Google Ads API para obtener credenciales OAuth y Developer Token
- [ ] **GA4** — Configurar proyecto en Google Cloud + habilitar Analytics APIs para credenciales OAuth
- [ ] **HubSpot** — Crear Private App en HubSpot (Settings > Integrations > Private Apps) para obtener PRIVATE_APP_ACCESS_TOKEN
- [ ] **Google Sheets** — Configurar proyecto en Google Cloud + OAuth para obtener Client ID, Client Secret y Refresh Token
- [ ] **Google Search Console** — Configurar proyecto en Google Cloud + habilitar Search Console API + crear Service Account (gratis)
- [ ] **Notion** — Crear integración en [notion.so/profile/integrations](https://www.notion.so/profile/integrations) para obtener token (gratis)
- [ ] **n8n** — Instalar y configurar servidor n8n (self-hosted, gratis) para automatización de flujos

## MCPs que requieren API key gratuita

- [ ] **Brave Search** — Registrarse en [api.search.brave.com](https://api.search.brave.com) para obtener API key (free tier: 2,000 consultas/mes)
